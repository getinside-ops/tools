About this tool

The DPI Checker analyses your image's resolution and tells you whether it's print-ready. DPI (dots per inch) determines print sharpness — 300 DPI is the professional standard for sharp output.

3 implementation approaches

A

Enhance ToolPageLayout (recommended)

Add category pill + "About" slot to the existing shared component. Update all 30 views to pass a category prop and fill the #about slot. PxToRemView gets migrated to ToolPageLayout.

✓ Single source of truth  ✓ Minimal risk  ✓ Clean pattern

B

New ToolAboutSection component

Same as A, but the "About" panel becomes its own dedicated ToolAboutSection.vue component with structured props (title, body, tips[ ]).

More structured, slightly more overhead. Worth it only if about content needs to vary in shape per tool.

C

Data-driven tools config

Store all tool metadata (title, category, about text) in a central tools.config.ts. Views consume from there. Fully centralized.

Cleanest architecture long-term, but over-engineered for the current scope. Big upfront change.
